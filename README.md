# Calendar Valley

A calendar interface for browsing AI events in 2025.

## Overview

This application displays upcoming AI conferences, workshops, and events. Events are loaded from a CSV file and presented in a weekly calendar view with OpenGraph preview fetching.

## Features

- Weekly calendar view of events
- Event details with OpenGraph previews
- Bookmark events for later reference
- Automatic scrolling to current week
- Progressive loading of event metadata

## Development

```bash
npm install
npm run dev
```

## Notes

Optimized for desktop viewing. Mobile users will see a recommendation to use a larger screen.
